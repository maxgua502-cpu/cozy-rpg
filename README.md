# Уютная RPG (рабочее название)

Одиночная браузерная RPG: топ-даун мир, пошаговый бой «дракон-дама-рыцарь», зверинец существ, лут-цикл, крафт, оффлайн-прогресс. Полностью локальная, без сервера.

- **Дизайн-документ (источник истины):** [docs/GDD.md](docs/GDD.md)
- **Правила разработки:** [CLAUDE.md](CLAUDE.md)

## Команды

| Команда | Что делает |
|---|---|
| `npm install` | установить зависимости (один раз) |
| `npm run dev` | запустить игру локально (откроется в браузере) |
| `npm run build` | собрать продакшн-версию в `dist/` |
| `npm run test` | юнит-тесты игровой логики (Vitest) |
| `npm run test:e2e` | smoke-тесты сцен (Playwright) |

Запуск на телефоне в локальной сети: `npm run dev` печатает адрес вида `http://192.168.x.x:5173` — открыть его на телефоне (ПК и телефон в одной Wi-Fi сети).

## Стек

Phaser 4 · Vite · vanilla JS (ES-модули) · Vitest · Playwright. Деплой — GitHub Pages (автоматически при пуше в `main`).

## Архитектура

Логика игры — чистые JS-модули в `src/systems/` (без Phaser, тестируются юнитами). Сцены Phaser в `src/scenes/` — только отображение и ввод. Контент (мобы, предметы, квесты) — JSON в `src/data/`. Подробнее — в [CLAUDE.md](CLAUDE.md).
